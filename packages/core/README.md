# @react-form-engine/core

Data-driven forms for React: schemas for vocabulary, code for logic.

Fields are plain typed data bound to your API model; the engine owns
parsing, validation, visibility, and serialization on top of
[TanStack Form](https://tanstack.com/form). Rendering is headless — pair
with [@react-form-engine/renderers-html](https://www.npmjs.com/package/@react-form-engine/renderers-html)
or bring your own renderer map.

- **[Documentation](https://xkirtle.github.io/react-form-engine/)**
- **[Live demo](https://xkirtle.github.io/react-form-engine/demo/)**
- [GitHub](https://github.com/xKirtle/react-form-engine)

```sh
npm install @react-form-engine/core @react-form-engine/renderers-html
```
